## Report - August 2021

## Overview

Over the month of August 2021 I fixed issues and merged pull-requests.
I worked on the sql-parser library and on the Debian package. I also wrote tests and improved code.

### First week (week number 31)

This week I was not working on the project but did some issue and pull-request triage.

### Second week (week number 32)

This week was focused on bug fixing and merging pull-requests.
I also improved documentation and worked on the sql-parser library.

### Third week (week number 33)

Over this week I focused on fixing bugs and writing improvements on the sql-parser library.
I also did some updates on Debian packaging. And improved our release script.
I wrote tests and reviewed pull-requests.

### Fourth week (week number 34)

This week started with some Docker repository fixes and improvements (upgraded the PHP version to PHP 8).
I did some testing improvements on the main repository with some bug fixes.
Also did code improvement and fixed security issues.
And I also worked on the Debian package to target the new backports branch.
Finally I wrote improvements to the sql-parser repository.
