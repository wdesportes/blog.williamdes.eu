# Consulting Services for phpMyAdmin

## Report - January 2021

## Overview

Over the month of January 2021 I worked everyday to catch up on the first week that was low on work hours.
I handled issues and pull-requests and made some code/performance improvements.
Did a lot of bug fixes and testing, also some GitHub actions migrations.

I would say that everything seems more in control now, issues are getting fixed and PRs merged.

### First week (week number 01)

This week was the final rush to make phpMyAdmin 5.0 land into Debian.
That was most of my work, I had things to do IRL.

### Second week (week number 02)

I started handling issues and pull-requests I had in my inbox, and made some Ubuntu PPA uploads.
On the `twig-i18n-extension` repository I did setup phpstan and migrate to GitHub actions.

### Third week (week number 03)

I finished my work on removing the Apache-2.0 files that where in our GPL-2.0 codebase.
Also migrated the main repository to GitHub actions and worked on a re-work of our version handling.
On the translation side I manually processed a lot of translations that where blocked because of code changes.
I sent the CVE-2021-21252 fix into Debian. And updated some PRs and tested some other ones.
Now we have a JS test suite using Jest !
Also worked on some bug fixes and found out some slowness in our code using xhprof.

### Fourth week (week number 04)

More tracking down the slowness users had reported and made the JS console async.
Fixed some bugs, did some PR and issue management.
Completely re-worked the docker repository test suite and removed TravisCI also there.
Wrote tests and re-based some contributor PRs and tested them.
In the background I upload a fix to our Ubuntu PPA.
