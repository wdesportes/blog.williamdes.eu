# Consulting Services for phpMyAdmin

## Report - March 2021

## Overview

Over the month of March 2021 my main focus was closing issues and pull-requests.
I also improved some code, wrote fixes and more tests. And upgraded our error reporting server to Cake PHP 4.

### First week (week number 09)

My focus of the week was migrating to Cake PHP 4 our error reporting server.
I also did fix issues and merge pull-requests. And wrote and improved tests to add more coverage.

### Second week (week number 10)

This week started with finishing the Cake PHP migration. I also worked on bug fixing and pull-request review.
On the main repository I made some code improvements and worked on the release script to get it clean.
Quite a bunch of issues where closed after the issue cleaning I did all the week.

### Third week (week number 11)

I made some error reporting server fixes and handled JS errors.
Also worked on our snapshot process that I improved by adding a prefix to the version.
I did some research about how I could speed up the client loading resources part.
And finally I worked on the error reporting modal and found way to load it at all times and get the issue fixed.

### Fourth week (week number 12)

This week I closed more issues and merged some pull-requests.
Worked on build reproducibility and wrote a CLI to write revision-info.php files.
Also did make bug fixes and removed a near to useless js lib.
